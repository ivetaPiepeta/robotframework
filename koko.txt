Installation and Setup on MacBook
Krok 1: Klonovanie repozitára
Klonuj repozitár na tvoj lokálny stroj:

bash
Kopírovať kód
git clone https://gitlab.websupport.sk/m-code/robotframework.git
cd robotframework
Krok 2: Vytvorenie virtuálneho prostredia
Vytvor virtuálne prostredie v priečinku projektu:

bash
Kopírovať kód
python3 -m venv venv
Krok 3: Aktivácia virtuálneho prostredia
Aktivuj virtuálne prostredie:

bash
Kopírovať kód
source venv/bin/activate
Krok 4: Inštalácia závislostí
Nainštaluj potrebné knižnice:

bash
Kopírovať kód
pip install robotframework
pip install robotframework-seleniumlibrary
pip install robotframework-requests
Krok 5: Inštalácia WebDriver
Ak používaš Chrome, nainštaluj chromedriver:

bash
Kopírovať kód
brew install chromedriver
Spustenie testov
Spusť testy pomocou Robot Frameworku:

bash
Kopírovať kód
robot EU_next.robot
Debugging
Ak narazíš na problémy s inštaláciou alebo spúšťaním testov, skontroluj nasledovné:

Uisti sa, že máš aktivované virtuálne prostredie.
Over, či sú všetky knižnice správne nainštalované:
bash
Kopírovať kód
pip list
Skontroluj, či je chromedriver nainštalovaný a dostupný v PATH.
